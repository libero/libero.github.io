eLife have handed over stewardship of Libero Editor to The Coko Foundation, and have moved focus away from development of Libero Reviewer and Libero Publisher.

You can now find the updated code repository for Libero Editor at https://gitlab.coko.foundation/libero/editor and continue the conversation on Coko's Mattermost chat server: https://mattermost.coko.foundation/

Libero Publisher and Libero Reviewer are available as software on GitHub and can be forked or reused with some effort as before although the eLife Team are no longer working on these projects.

For more information on why we're doing this read our latest post on our updated technology direction: https://elifesciences.org/inside-elife/daf1b699/elife-latest-announcing-a-new-technology-direction
